### 2.3 数据库表

新增数据表 `gesture_config`，存储手势配置参数：

```sql
CREATE TABLE gesture_config (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    gesture_name TEXT NOT NULL UNIQUE,
    display_name TEXT NOT NULL,
    description TEXT,
    enabled INTEGER NOT NULL DEFAULT 1,
    action TEXT NOT NULL,
    sensitivity_threshold INTEGER DEFAULT 50,
    hold_time INTEGER DEFAULT 500,
    created_at TEXT NOT NULL DEFAULT(datetime('now')),
    updated_at TEXT NOT NULL DEFAULT(datetime('now'))
)
```

## 四、5个手势功能实现

| 手势 | 动作 | 实现方式 |
|------|------|----------|
| **上划** | 返回上一页 | `window.history.back()` |
| **下划** | 刷新当前页面 | 根据路由判断，调用对应刷新函数 |
| **张开手掌** | 打开快捷工具面板 | 创建侧边栏面板，显示系统状态 |
| **竖起大拇指** | 跳转智能问数 | `window.location.href = '/chat'` |
| **握拳** | 截图保存 | 使用html2canvas截图并下载 |

## 六、使用方法

### 6.1 访问演示页面

启动服务器后，访问：`http://localhost:1086/gesture/demo`

### 6.2 启用手势识别

1. 点击页面右下角的紫色圆形按钮
2. 允许浏览器访问摄像头
3. 手势识别自动启动

### 6.3 使用手势

- **上划**：手掌向上滑动 → 返回上一页
- **下划**：手掌向下滑动 → 刷新当前页面
- **张开手掌**：五指张开 → 打开快捷工具面板
- **竖起大拇指**：竖起大拇指 → 跳转到智能问数
- **握拳**：握紧拳头 → 截图保存

### 6.4 配置管理

访问后台：`http://localhost:1086/admin/gesture`

可以：
- 启用/禁用特定手势
- 调整识别灵敏度
- 调整手势保持时间

## 八、注意事项

### 8.1 依赖库

截图功能需要html2canvas库，请下载并部署到：
`/app/static/dist/html2canvas.min.js`

### 8.2 浏览器兼容性

支持现代浏览器：
- Chrome（推荐）
- Firefox
- Edge
- Safari

### 8.3 摄像头权限

首次使用需要允许浏览器访问摄像头权限。

### 8.4 性能建议

手势识别会占用一定CPU资源，建议：
- 在需要时开启
- 光照充足的环境下使用
- 简化背景以提高识别准确率
